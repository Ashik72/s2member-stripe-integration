== Description ==

s2Member Stripe Integration plugin connects your s2Member to Stripe and lets you charge for your memberships. Now you can add Stripe as a payment gateway to s2Member.

== Installation ==

1. Upload 's2member-stripe-integration.zip' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can view the plugin settings in the 's2Member Stripe' menu .
4. Visit the stripe.com and create a new account if you don't have one.
5. Please, login and you will find 'Account Settings' under 'Your Account'.
6. Navigate to 'API Keys' tab.
7. Copy and paste the test keys i.e Publishable Key and Secret Key for testing. 
8. Enter the amount to be charged for the featured job posting.
9. We can disconnect the plugin from stripe if you don't want to disable it temporarily.


== Frequently Asked Questions ==

= I don't have a stripe account =

You can create one in stripe.com its free to register.

= What are the currencies supported?  =

The currency support is based on stripe. Under stripe -> account settings -> genral tab, you can change the country to match your currency.

= How can i test its working ? =

 Use test keys and the credit card number 4242 4242 4242 4242 with a future date.
